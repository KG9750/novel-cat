# 世界观规则

版本：2026-04-28

## 一、基础规则

1. 灵气早已渗入城市，不是突然复苏。
2. 普通人看见的是失眠、暴躁、噩梦、怪病和意外事故。
3. 异能局看见的是污染、影斑、异常物、觉醒者和异能兽。
4. 城市恶意会沉积成影斑。压迫越久、罪越重、执念越深，影斑越厚。
5. 异常不会凭空出现。每个异常背后必须有现实动因、利益链或情绪土壤。

## 二、异常三类

觉醒者：

1. 人类被异能污染后稳定觉醒。
2. 能力可控，但会受情绪、污染浓度和训练影响。
3. 非法觉醒者可能成为副本反派或黑市打手。

异常物：

1. 被执念、恶意、灾厄浸泡后产生规则。
2. 有触发条件、使用代价和污染扩散方式。
3. 例如红眼合同、偷脸镜、催债影印。

异能兽：

1. 动物受异能影响后进化。
2. 保留动物本能，也可能拥有高智商。
3. 煤球是 S 级影噬兽，能吞更深层的恶意与异常。

## 三、影斑规则

影斑来源：

1. 犯罪者长期作恶。
2. 异常物污染使用者。
3. 凶案现场、压迫关系、非法实验场所。
4. 被人为喂养的城市恶意节点。

影斑表现：

1. 普通人看不见，只感到压抑、倒霉、易怒。
2. 林澈通过技能毛球或煤球反应间接识别。
3. 煤球能嗅到、咬住、吞噬。

影斑粗分级：

1. 薄斑：日常恶意残留，能提供线索或小额影晶。
2. 厚斑：长期作恶沉积，可能吐出关键记忆碎片。
3. 影巢：被持续喂养的污染节点，会制造副本或异常规则。
4. 污染源：异常链条核心，清理代价高，常牵出幕后组织。

影斑限制：

1. 影斑不是罪证本身，只是恶意残留。
2. 煤球吐出的记忆碎片只能来自真实痕迹。
3. 影斑越强，返还越高，煤球代价越重。

## 四、影噬返还

煤球吞噬影斑后，可能返还四类东西：

1. 影晶：可交易、升级、入药、给异能装备供能。
2. 记忆碎片：画面、声音、坐标、密码、证据线索。
3. 技能毛球：夜视、兽语、危险预感、影遁等短时能力。
4. 异常残页：拼出煤球来历、林晚污染和幕后组织。

返还原则：

1. 小恶意出线索，大污染出高价值返还。
2. 返还物不能凭空解决所有问题。
3. 返还物必须带来新风险。
4. 同类返还不能频繁重复，要有用途变化。

影晶价格简表：

1. 普通影晶：几千到几万，看纯度和用途浮动。
2. 高纯影晶：数万到十万，可用于装备、入药或稳定仪式。
3. 特殊影噬产物：十万起，因来源稀少，会被黑市和研究所盯上。

## 五、代价规则

煤球代价：

1. 吞噬后嗜睡、虚弱、发冷。
2. 强污染可能导致失控、暴食、敌我不分。
3. 长期吞噬会唤醒夜阙旧记忆。

林澈代价：

1. 使用技能毛球会疲惫。
2. 可能短暂被恶意情绪污染。
3. 频繁接触异常会被异能局和黑市记录。

技能毛球反噬简表：

1. 夜视：眼痛、畏光、短时残影。
2. 兽语：耳鸣、动物情绪残留、难以立刻入睡。
3. 危险预感：恐惧放大、心跳失衡、容易误判普通风险。

交易代价：

1. 影晶交易会留下异能波动。
2. 黑市能帮忙变现，也会泄露风声。
3. 异能局能提供合法额度，也会加强监管。

## 六、异能局规则

职能：

1. 监测异常。
2. 管控觉醒者、异常物和异能兽。
3. 清理低级影斑。
4. 隐瞒普通社会无法承受的信息。

三线：

1. 外勤杨小北：现场处置、临时授权、对林澈担保。
2. 档案权限：决定林澈能看到多少旧案和夜阙资料。
3. 研究所秦衡：收容、检测、研究优先，容易和外勤冲突。

限制：

1. 不能公开承认所有异常。
2. 不能放任 S 级异能兽自由行动。
3. 内部有外勤、研究所、监察、档案等不同立场。
4. 杨小北代表现场秩序，秦衡代表研究所收容逻辑。

## 七、黑市规则

存在理由：

1. 异能局监管有盲区。
2. 影晶和异常物有高利润。
3. 普通人和觉醒者都可能绕开官方求助。

运作方式：

1. 情报、悬赏、检测、交易分开进行。
2. 陈野只卖消息，不保证安全。
3. 黑市重视规矩，但规矩服务利益。

风险：

1. 假情报。
2. 钓鱼交易。
3. 影晶波动暴露煤球。
4. 幕后组织借黑市回收影噬兽产物。

## 八、设定使用原则

1. 每章只新增一个主要设定点。
2. 新设定必须马上制造麻烦或帮助破局。
3. 不提前解释三年前旧案。
4. 不让异能局成为万能后勤。
5. 不让黑市成为万能答案库。
