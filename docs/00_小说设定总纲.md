# 小说设定总纲

## 书名

《我捡的流浪猫，竟是 S 级异能兽》

书名关键词：流浪猫、S 级、异能兽。读者一眼能看到萌宠、等级爽点、都市异能三个核心卖点。

## 一句话卖点

失业那晚，林澈捡回一只快冻死的流浪黑猫。第二天，拖欠他工资的主管被自己的影子吐出了贪污证据，而黑猫叼着一枚价值十万的异能晶核跳上他的床。从那以后，林澈发现：这只猫能吞噬世间恶意，并把别人的罪，变成他的逆袭资本。

## 核心爽点

主角遇到现实困境：欠薪、被裁、房租、妹妹医药费、被栽赃。

黑猫发现对方身上的“恶意污染”。

主角带猫接触事件，判断谁在撒谎，找到恶意源头。

黑猫吞噬污染，吐出证据、晶核、技能毛球、情报碎片。

主角利用这些收益完成反击，同时一步步卷入异能世界。

## 类型定位

都市异能、萌宠、现实逆袭、轻悬疑副本、轻喜剧。

气质：前期贴近生活，主打“坏人藏得越深，猫吐得越狠”；中期进入异能局、黑市、异常事件；后期揭开黑猫真实身份和主角体质。轻喜剧主要来自煤球的猫性反差、林澈和异能局的身份错位、普通生活与异常事件的碰撞，不削弱主线危机。

## 世界观

灵气不是突然复苏，而是早已渗入城市缝隙。

普通人看见的是失眠、暴躁、噩梦、怪病、意外事故。

异能局看见的是三类异常：

1. 觉醒者：人类被异能污染后稳定觉醒，拥有可控能力。
2. 异常物：被执念、恶意、灾厄浸泡后产生规则的物品。
3. 异能兽：动物受异能影响后进化，常保留本能，也可能拥有高智商。

城市里的恶意会沉积成“影斑”。影斑通常依附在犯罪者、异常物、凶案现场、长期压迫关系中。影斑越厚，事件越危险，吞噬后的返还越高。

普通觉醒者只能清除低级影斑。黑猫能吞噬更深层的“恶意与异常”，这是它被异能局、研究所、黑市同时盯上的原因。

## 黑猫能力

能力名：影噬返还。

表现：黑猫能嗅到人、物、地点上的恶意污染，吞噬后吐出返还物。

返还类型：

1. 影晶：异能污染凝成的晶核，可交易、升级、入药、作为异能装备能源。
2. 记忆碎片：与恶意源头相关的画面、声音、坐标、密码、证据线索。
3. 技能毛球：一次性短时能力，可能是夜视、敏捷、危险预感、影遁、兽语。
4. 异常残页：中后期出现，能拼出黑猫来历和大型阴谋。

限制：

1. 黑猫不能凭空制造证据，只能吐出真实恶意留下的痕迹。
2. 吞噬越强的污染，黑猫越容易虚弱、嗜睡、失控。
3. 主角使用技能毛球会承受反噬，轻则疲惫，重则被恶意影响情绪。
4. 影晶不能随意出售，交易会留下异能波动，引来异能局或黑市。
5. 黑猫愿不愿意出手，取决于林澈的选择。它不是工具，而是伙伴。

## 主角

姓名：林澈。

年龄：27。

身份：普通公司数据审核员，开局被裁、被欠薪、被栽赃背锅。

现实压力：房租到期，妹妹林晚长期住院，前公司拖欠工资，主管准备把财务黑账推到他身上。

性格底色：温和、能忍、做事细，但不是圣父。他只是被生活磨得习惯先退一步。

成长线：从被动挨打的底层社畜，成长为能判断异常、布局反击、保护伙伴的“城市异常清道夫”。

核心优势：

1. 数据敏感，擅长从碎片证据里还原真相。
2. 心软但不糊涂，愿意救弱者，也敢对恶人补刀。
3. 拥有“净土体质”，能稳定黑猫吞噬后的暴走。

## 黑猫

小名：煤球。

真实代号：夜阙。

表面：瘦巴巴的流浪黑猫，挑食、记仇、爱踩键盘，喜欢睡在林澈枕边。

真实身份：S级异能兽“影噬兽”，曾是某次大型异常事件里的核心生物。

性格：高冷，嘴硬，护短。它不会说人话，但会用眼神、爪子、尾巴和吐出来的东西表达态度。

与主角关系：前期像主宠，中期像搭档，后期反转为互相拯救。它选择林澈，不是因为林澈强，而是因为林澈身上没有恶意，并且能让它在吞噬污染后保持清醒。

## 重要角色

林晚：林澈妹妹，长期住院。她不是单纯拖油瓶，后期会觉醒“共鸣”类能力，能听见异能兽情绪。

赵启明：林澈前主管。现实线第一名反派，拖欠工资、贪污项目款、伪造数据栽赃林澈。用来完成第一次打脸。

杨小北：异能局外勤监察员。冷静、专业、行动力强，习惯用规则判断风险。最初怀疑林澈非法饲养S级异能兽，甚至一度想把煤球带回异能局登记管控。后期成为林澈的稳定合作对象，逐渐被林澈的判断力、责任感和对煤球的尊重吸引。感情线走慢热路线：她先是欣赏，再是信任，最后主动追求林澈；她的追求不撒娇卖弱，而是用并肩作战、替他担责、给他选择权来靠近。

许清妍：林澈大学时期一直暗恋的对象，也是他心里“求而不得”的白月光。她温柔、体面、距离感强，曾在林澈最狼狈的时候帮过他，所以林澈对她有长期滤镜。剧情推进后，林澈发现她身上寄生着一种不好的异常物，暂名“愿虫”。愿虫会放大旁人的好感、愧疚和保护欲，也会慢慢吸走亲近者的运势与情绪。许清妍是否知情先保持悬念。随着林澈发现这层异常，他对许清妍的感情从执念、心疼、怀疑逐渐转为清醒，感情天平开始向杨小北倾斜。

周言：林澈的铁哥们儿，大学室友，现在是自由摄影师兼本地生活探店博主。嘴碎、讲义气、胆子不大但关键时刻不掉链子。前期负责给林澈提供现实层面的帮助，比如借钱、藏猫、跑腿、打听消息；中期因为拍到异常事件边角料，被迫卷入异能世界。他不是战力担当，而是信息、烟火气和轻喜剧担当，也能从普通人的角度提醒林澈别被异能世界吞掉正常生活。

陈野：黑市情报贩子，嘴碎，贪财，胆小但讲规矩。负责提供悬赏、副本入口、影晶交易渠道。

秦衡：异能局研究所负责人，表面温和，实则想把煤球带回实验室。他不是单纯坏人，而是相信“危险必须被关起来”。

## 反派层级

前期：现实小人。

主管、房东、黑心宠物医院、栽赃同事、骗保中介。

中期：异常事件背后的获利者。

黑市商人、邪性觉醒者、污染物收藏家、非法研究组织。

后期：制造影斑生态的人。

某些机构利用城市恶意喂养大型异常，试图人工制造可控S级异能兽。煤球不是意外，而是幸存者。

## 长线悬念

1. 煤球为什么会重伤流落街头？
2. 为什么只有林澈能安抚煤球？
3. 影晶到底是谁在高价回收？
4. 林澈妹妹的病，是普通疾病，还是异能污染的早期表现？
5. 异能局究竟是在保护城市，还是在隐瞒更大的灾难？

## 写作原则

1. 每个副本都要先落到现实痛点，再打开异能真相。
2. 猫负责发现和吞噬，林澈负责判断、布局、谈判、承担后果。
3. 证据、钱、能力都要服务于主角逆袭，不做纯设定展示。
4. 爽点要即时兑现，但每次兑现都引出更大的危险。
5. 黑猫要可爱、有脾气、有动作，不要只当挂件。
6. 前期能力低显性释放：先像眼花和可验证疑点，后续再逐步确认影晶、技能和 S 级强度。
