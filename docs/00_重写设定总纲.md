# 《我的宠物猫竟是只异能兽》重写设定总纲

版本：外卖小哥新骨架

## 重写边界

本版从零重写，不沿用旧版职场流骨架。

明确废弃：

- 数据审核员、公司裁员、主管甩锅、财务黑账。
- 电脑旧系统、缓存、权限、日志、导出路径等主要破局方式。
- 用流程解释推动剧情的高密度职场口吻。

保留核心读者承诺：

- 普通低位男主救下一只黑猫。
- 黑猫不是普通宠物，而是高危异能兽。
- 黑猫能嗅到恶意污染，吞掉污染后吐出真实痕迹。
- 男主不是躺赢，必须靠自己的观察、跑动、取证和选择完成反击。

## 类型定位

都市异能、萌宠、外卖小哥逆袭、轻悬疑副本、强 CP 并肩、轻喜剧。

气质：前期贴近生活，读者先看懂“外卖小哥被恶意投诉、账号封禁和医院押金压住”，再看到“猫能把坏人藏起来的恶意咬出来”。中期进入异常订单、异能局、黑市和城市裂隙。后期揭开煤球真实身份和林澈为什么能安抚它。

## 一句话卖点

外卖小哥林澈接到雨夜最后一单，普通宠物急送却把他引进废弃仓库。走投无路时，他捡回一只快冻死的黑猫，也因此被客户投诉拒送、擅自带走宠物，账号面临永久封禁。第二天，黑猫咬碎客户影子里的恶意，吐出被调包的订单小票和一枚黑晶。

从此，林澈发现：这只猫能吞掉城市里的恶意污染，把坏人藏起来的罪，变成他翻身的证据和资本。

## 主角

姓名：林澈。

年龄：27。

职业：同城急送众包骑手，夜间常接医院、宠物医院、旧小区和商圈加急单。

现实压力：

- 妹妹林晚长期住院，押金快见底。
- 房租到期，电瓶租赁费和车架维修费没结清。
- 平台账号一旦封禁，他当天就断收入。
- 站点只看超时率和投诉率，不关心他有没有被栽赃。

核心优势：

- 熟悉江城路线、门禁、后门、监控死角和夜间小路。
- 记得住取餐号、楼栋门牌、客户语气和异常细节。
- 会拍照留痕、录音自保、找门卫、护士、店员核实。
- 有底层人的忍耐，但不是没脾气；被逼到妹妹和猫身上时，会反击。

成长线：

被订单和投诉压着跑的人，成长为能在城市异常里判断风险、保护伙伴、主动设局的异常配送人。

## 黑猫

小名：煤球。

真实代号：夜阙。

表面：被雨淋透的瘦黑猫，挑食、记仇、爱钻保温箱，不说人话。

真实身份：S 级异能兽，曾从某个大型异常回收项目中逃出。

能力：影噬返还。

表现：

- 能闻到人、物、地点上的恶意污染。
- 能吞掉真实存在的影斑。
- 吞噬后吐出黑晶、残缺小票、记忆碎片、一次性技能毛球、旧编号残片。

限制：

- 不能凭空制造证据，只能吐出恶意留下的真实痕迹。
- 吞得越深，身体越冷，越容易嗜睡、失控或短暂认不出林澈。
- 如果林澈为了钱强迫它吞噬，它会拒绝，甚至伤到林澈。
- 它愿意出手，不是因为林澈命令它，而是因为林澈救它、护它，也愿意为选择付代价。

## 女主/感情线

姓名：杨小北。

身份：异能管理局江城外勤，负责异常订单、异能兽走私和城市污染点处置。

关系定位：

- 前期：她把林澈当风险对象，要求交出煤球登记。
- 中期：她发现林澈不是乱来的普通人，而是能在现场保护煤球也保护无辜者的人。
- 后期：她从监管者变成并肩者，愿意为林澈和煤球承担程序代价。

感情线原则：

- 慢热，不撒糖硬凑。
- 她先认可林澈的判断和责任，再靠近这个人。
- CP 爽点来自互相站队、互相救场、共同对抗规则，不靠降智误会。

## 重要配角

林晚：林澈妹妹，住在江城三院。她不是单纯病弱拖累，后期能感知异能兽情绪，是煤球失控时的重要锚点。

周言：林澈骑手朋友，白天送货，晚上摆摊修车。嘴碎、胆小但讲义气，提供普通人支援。

陈野：黑市跑腿中介，专接不能上平台的异常委托。贪财，怕死，有规矩。

秦衡：异能兽研究所负责人，主张高危异能兽必须回收。他不是脸谱反派，但会为了“安全”牺牲煤球的自由。

## 前期反派层级

第一层：现实恶意。

- 恶意投诉客户。
- 黑心站长。
- 倒卖宠物药的中间人。
- 利用外卖骑手背锅的商家。

第二层：异常订单。

- 会自己改地址的订单。
- 门牌号重复的旧楼。
- 吃掉差评的红色小票。
- 能让人迷路的外卖柜。

第三层：异能兽回收链。

- 黑市收购影晶。
- 异能兽药品走私。
- 研究所旧编号。
- 夜阙回收程序。

## 长线悬念

1. 煤球为什么会出现在雨夜最后一单的地址附近？
2. 为什么煤球只让林澈抱，而不攻击他？
3. 林晚的病为什么会和异常订单产生共鸣？
4. 谁在利用外卖骑手运输异能兽污染物？
5. 异能局保护城市，还是也在掩盖夜阙旧案？

## 单章写作规则

- 每章开头直接进入事件，不写长篇设定。
- 每章给林澈一个具体配送、救人、取证、保护猫或保护妹妹的任务。
- 破局尽量用看得见的动作：跑楼、拍照、问门卫、翻小票、核对门牌、追电瓶车、抱猫躲人。
- 煤球不说人话，用爪子、尾巴、耳朵、叼东西、吐东西表达。
- 章尾必须留一个具体钩子：电话、订单、爪印、门牌、证件、监测环、黑晶。
