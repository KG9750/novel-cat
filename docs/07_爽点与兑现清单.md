# 爽点与兑现清单

版本：2026-04-28

## 一、核心爽点公式

> 坏人留下恶意，煤球吐出证据，林澈亲手反击。

爽点必须具体，优先落到：

1. 钱到账。
2. 证据成链。
3. 能力有用。
4. 身份升级。
5. 猫护主。
6. 反派当场露馅。

## 二、爽点分类

现实逆袭：

1. 追回欠薪。
2. 洗清栽赃。
3. 反击威胁妹妹的人。
4. 搬离危险出租屋。
5. 给妹妹补上医药费。

萌宠爽：

1. 煤球嫌弃贫穷伙食。
2. 煤球踩键盘触发线索。
3. 煤球护住林澈。
4. 煤球用爪印按协议。
5. 煤球虚弱时只认林澈。

吐宝爽：

1. 黑晶第一次出现。
2. 技能毛球第一次使用。
3. 记忆碎片拼出证据链。
4. 异常残页牵出林晚。
5. 高级影晶引发黑市报价。

打脸爽：

1. 主管当场露馅。
2. 房东合同反噬。
3. 黑心医院被曝光。
4. 财务总监转账链被拍出来。
5. 赵启明发现自己也只是替罪羊。

升级爽：

1. 林澈从普通人到临时协作者。
2. 掌握影晶交易规则。
3. 学会使用技能毛球。
4. 被异能局重新评估。
5. 获得合法交易额度。

## 四、爽点密度规则

单章：

1. 至少有一个小爽点或强线索。
2. 不能只有受压，没有反弹。
3. 受压章节也要给读者希望，例如猫反应、证据碎片、反派破绽。

三章：

1. 至少完成一次承诺兑现。
2. 至少出现一次煤球不可替代的行为。
3. 至少让林澈做一次主动选择。

十章：

1. 至少完成一个副本。
2. 至少获得一类明确收益。
3. 至少升级一次主线危险。

## 五、禁止空爽

1. 只写“众人震惊”，但没有可验证证据。
2. 只写“影晶很贵”，但没有价格、交易或用途。
3. 只写“林澈很聪明”，但不展示推理过程。
4. 只写“煤球很强”，但不写代价。
5. 只让反派嘴硬，不让反派付出实际损失。
