# 写作风格卡

版本：2026-04-28

## 一、整体手感

关键词：

顺口、轻快、现实、有画面、轻幽默、强钩子。

一句话：

> 现实有雨水和账单，异能先藏在灰黑水痕和猫的怪反应里，猫负责把沉重的东西踩出一点笑声。

## 二、叙述节奏

1. 短段落，少长句。
2. 动作先行，解释后置。
3. 先让读者看见，再让读者理解。
4. 设定只在角色需要时出现。
5. 每个场景尽快进入冲突。

不写：

1. 长篇世界观说明。
2. 连续心理独白。
3. 华丽但不推进的景物描写。
4. 角色互相复述读者已知信息。

## 三、幽默手感

幽默来源：

1. 煤球的高冷和猫性反差。
2. 林澈穷得真实，但吐槽克制。
3. 异能局严肃流程被猫打乱。
4. 周言和陈野负责嘴碎，但不能抢戏。

示例方向：

1. 煤球吐完价值十万的影晶，先嫌弃地看林澈的泡面。
2. 杨小北拿检测仪，煤球装普通猫，下一秒踩碎仪器。
3. 陈野开价很狠，听见 S 级两个字立刻想退单。

禁忌：

1. 不堆网络段子。
2. 不在生死危机里硬贫。
3. 不把主角写成油嘴滑舌。
4. 不反复用同一个猫梗。

## 四、场景质感

每个重要场景放三类细节：

1. 现实细节：潮湿墙皮、旧工牌、催租纸条、医院缴费短信、消毒水味。
2. 异常细节：影斑、红印、黑气、镜中延迟、无声盯视。
3. 猫的反应：炸毛、伏低、甩尾、舔爪、叼走、踩键盘。

写法顺序：

现实正常 -> 一个地方不对 -> 煤球先发现 -> 林澈跟进判断。

前期克制：

1. 第一章少用大范围异象，不写“全场影子变化”。
2. 异常先像眼花、屏幕坏点、潮湿水痕、猫的反常反应。
3. 影晶价值、S 级强度、完整技能效果后移，让能力慢慢释放。

## 五、人物语言

林澈：

1. 语气克制，少喊口号。
2. 被逼急时话短，行动多。
3. 偶尔冷幽默，但不轻浮。

煤球：

1. 不说人话。
2. 用动作表达态度。
3. 动作要准，少拟人化解释。

杨小北：

1. 句子简洁，职业化。
2. 先讲规则，再看人。
3. 情绪不外放，但会用行动承担。

周言：

1. 嘴碎、生活化。
2. 怂得真实，义气也真实。
3. 负责把异常翻译成普通人的反应。

陈野：

1. 会算账，会试探。
2. 怂但守规矩。
3. 信息永远不白给。

秦衡：

1. 温和、理性、有压迫感。
2. 话里常有“风险”“控制”“必要代价”。
3. 不暴躁，越平静越危险。

## 六、人物描写

人物要鲜活，不能只是推动情节的工具人。每个重要角色出场时，至少让读者看到他在当前压力下的一个真实反应：想要什么、怕什么、忍了什么、选择了什么。

心理活动写法：

1. 心理活动要短而准，贴着当下动作，不写长篇自我剖析。
2. 先写外部刺激，再写身体反应，再写一句内心判断或压住的情绪。
3. 心理活动要暴露性格：林澈会先算风险和证据，周言会先怕再帮忙，杨小北会先想规则和后果。
4. 同一情绪不要反复说明，用动作替换重复心理，例如握紧手机、删掉消息、移开视线、把话咽回去。

行为细节写法：

1. 人物性格要落在小动作里：林澈备份、截图、压住怒火；杨小北先看现场风险；陈野先算价格；周言嘴上怂但脚没退。
2. 每个配角出场不只交付信息，还要带出自己的利益、顾虑、习惯或情绪。
3. 反派不能只负责威胁和震惊，要让读者知道他为什么这么做、怕什么败露、靠什么获利。
4. 猫的动作也要有情绪和选择，不能只是自动触发剧情按钮。

语言差异：

1. 不同人物的句子长度、用词和关注点要不同。
2. 林澈话短、克制、带一点冷幽默；赵启明软硬兼施，常把威胁包装成“为你好”；杨小北职业化，先规则后判断；周言生活化、嘴碎；陈野会试探和算账；秦衡温和但有控制感。
3. 避免所有人都用同一种解释腔、吐槽腔或狠话腔。
4. 对话不能只复述剧情信息，必须带关系、试探、遮掩、压迫或情绪变化。

人物弧光：

1. 每章至少让一个核心人物发生微变化：知道新信息、做出新选择、付出新代价、关系推进或底线被触碰。
2. 林澈的弧光是从忍到查、从查到布局、从布局到承担后果，不能只靠煤球替他赢。
3. 煤球的弧光是从警惕被收留，到选择配合，再到信任林澈；它的每次出手都要能看出态度。
4. 杨小北的弧光是审查 -> 观察 -> 有条件信任，不要跳成无条件帮助。
5. 许清妍、秦衡这类复杂角色要保留自洽逻辑，不要过早写成单薄好人或坏人。

## 七、句式模板

压迫句：

> 这句话比雨更冷。

异常句：

> 煤球没有叫，只是盯着那道影子，瞳孔一点点收成细线。

爽点句：

> 林澈把录音点开，赵启明的声音从手机里清清楚楚地响了起来。

章尾句：

> 煤球这次吐出来的不是影晶，而是一张写着林晚名字的残页。

## 八、修稿检查

1. 这一页有没有猫的存在感？
2. 这一章有没有具体证据或具体收益？
3. 这一段能不能删掉解释，改成动作？
4. 这个笑点会不会削弱危机？
5. 章尾是不是让人想点下一章？
6. 这一章有没有至少一个人物的内心反应或选择变化？
7. 这段对话换个人说会不会完全一样？如果一样，就要重写口吻。
8. 这个角色出场是否只在交付信息？如果是，补一个欲望、顾虑、习惯或代价。
