# 读者承诺与卖点

版本：2026-04-28

适用：《我捡的流浪猫，竟是 S 级异能兽》

## 一、读者承诺

读者点进本书，是为了看：

1. 一个被现实欺负的普通人，靠猫和脑子一步步翻身。
2. 一只高冷、护短、有脾气的黑猫，把坏人藏起来的罪吐出来。
3. 每个现实痛点背后，都能挖出更深的异能真相。
4. 每次爽点兑现后，都有更大的危险和悬念跟上。

不能背叛的承诺：

1. 不让林澈长期窝囊。可以受压，但必须反击。
2. 不让煤球变成无成本外挂。它强，但会虚弱、选择、失控。
3. 不写纯设定展示。证据、钱、能力都要服务逆袭。
4. 不把故事写成沉重苦情。现实要疼，猫味和轻喜剧要在。

## 二、核心卖点

一句话卖点：

> 失业那晚，林澈捡回一只流浪黑猫。从此，坏人的恶意会被猫吞掉，罪证、晶核和技能毛球会被它吐到林澈面前。

标题卖点：

1. 流浪猫：萌宠入口，降低异能题材门槛。
2. S 级：等级爽点，暗示猫的真实身份和危险价值。
3. 异能兽：都市异能大世界入口。

前期强卖点：

1. 欠薪打脸：现实痛点强，读者容易代入。
2. 猫吐证据：金手指新鲜，画面直观。
3. 影晶值钱：钱的爽点明确。
4. 异能局上门：主线从职场反击升级为城市异常。

中期强卖点：

1. 宠物医院副本：救猫救动物，情绪浓度高。
2. 黑市交易：影晶价值扩大，风险扩大。
3. 妹妹污染：亲情线和主线悬念合流。
4. S 级档案：煤球从萌宠升级为核心谜题。

后期强卖点：

1. 煤球来历：它不是意外，是幸存者。
2. 林澈体质：他不是被动绑定，而是能稳定煤球的关键。
3. 影斑生态：城市恶意被人为喂养。
4. 互相拯救：林澈救煤球，煤球也把林澈从生活泥潭里拖出来。

## 三、简介素材

短简介：

> 失业、欠薪、被栽赃的那晚，林澈捡回一只快冻死的流浪黑猫。第二天，主管的影子吐出了贪污证据，黑猫又叼回一枚价值十万的异能晶核。林澈这才发现，它能吞噬世间恶意，把别人的罪变成他的反击资本。

长简介：

> 林澈只是个普通数据审核员，被裁、欠薪、背黑锅，妹妹的住院费还压在眼前。雨夜里，他救下一只瘦得只剩一双金眼睛的黑猫。黑猫不说话，挑食、记仇、爱踩键盘，却能闻到人身上的恶意污染。它吞掉影斑，吐出证据、晶核、技能毛球和记忆碎片。从欠薪主管到黑心医院，从异常合同到城市影巢，林澈带着猫一步步反击，也一步步发现，这只流浪猫，竟是异能局和黑市都在寻找的 S 级异能兽。

推荐标签：

都市异能、萌宠、流浪猫、现实逆袭、轻悬疑、副本、打脸、异能局、黑市、轻喜剧。

## 四、读者期待管理

开篇 1 章给：

1. 林澈的现实困境。
2. 煤球登场。
3. 第一次低显性异常返还。
4. 明确章尾威胁。

前三章给：

1. 赵启明栽赃链条。
2. 煤球能力可信度。
3. 林澈第一次主动反击。
4. 杨小北和异能局入口。

卷一给：

1. 欠薪和栽赃解决。
2. 第一桶金到手。
3. 异能局协作者身份。
4. 煤球 S 级身份露出一角。
5. 妹妹污染和幕后回收影噬兽产物的长线钩子。
