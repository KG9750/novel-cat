# 本书写作指南

版本：2026-04-28

适用：《我捡的流浪猫，竟是 S 级异能兽》

## 一、核心定位

类型：都市异能、萌宠、现实逆袭、轻悬疑副本、轻喜剧。

主读感：顺、爽、解气、好笑、有猫、有钩子。

一句准绳：

> 现实先压人，煤球闻恶意，林澈找证据，坏人被反噬。

全书不能偏离三件事：

1. 现实痛点要真：欠薪、栽赃、房租、医药费、黑心机构。
2. 异能入口要早但低显性：先让猫发现异常痕迹和可疑返还物，再逐章释放影晶、技能、情报。
3. 主角反击要主动：煤球提供入口，林澈判断、布局、谈判、承担后果。

## 二、番茄写法硬规则

1. 开篇三章必须完成：受压、救猫、低显性返还、查证、打脸、异能局上门。
2. 每章 2000-2600 字左右，短段落，快转折，少空话。
3. 前 300 字进入冲突或异常，不能长铺背景。
4. 每章至少有：一个问题、一个猫动作、一个林澈主动选择、一个章尾钩子。
5. 爽点即时兑现，不能连续多章只让主角受气。
6. 每 3-5 章有小高潮，每 8-12 章完成一个副本或阶段性收益。
7. 设定跟着事件走，不提前讲课。
8. 章尾钩子要具体：谁来了、什么证据变了、猫吐出了什么、谁盯上了它。

## 三、风格基调

本书借鉴方法，不模仿句子。

1. 轻松感：来自煤球的猫性反差、林澈的克制吐槽、异能世界和普通生活的错位。
2. 副本感：每个异常都有规则、代价和破解点，不靠猫一口吞完。
3. 场景感：现实细节先落地，再给异常细节，最后让煤球反常。
4. 爽文感：小目标连续达成，收益递进，风险升级。
5. 大世界感：异能局、黑市、研究所、S 级档案逐步露出，不抢前期主线。

前期异能释放原则：

1. 第一章只给“像眼花但能验证”的异常，不写大范围异象。
2. 返还物先像怪东西，价值和规则放到第 4 章以后确认。
3. 毛球只给碎片线索，不能像完整回放或神谕。
4. 每次能力露出都必须落回现实证据，让普通读者先信林澈，再信异能。

## 四、剧情公式

标准副本链：

> 现实痛点 -> 影斑出现 -> 林澈发现矛盾 -> 煤球确认恶意 -> 试探规则 -> 主角布局 -> 煤球吞噬 -> 返还物出场 -> 反击兑现 -> 更大危险露头。

每个副本必须回答：

1. 坏人靠什么获利？
2. 异常规则是什么？
3. 林澈不用蛮力，怎么破局？
4. 煤球吐出的东西带来什么收益和麻烦？

## 五、黄金前三章

第 1 章：流浪黑猫

1. 写清林澈被裁、欠薪、妹妹医药费、被栽赃。
2. 救猫体现林澈底色：穷，但心不坏。
3. 煤球第一次只露出异常痕迹、黑色硬块和毛球，画面要克制，让读者觉得“怪，但还能跟”。
4. 章尾用陌生电话钩住后续。

第 2 章：猫吐出来的证据

1. 林澈验证毛球不是幻觉。
2. 他主动查旧系统、缓存、权限记录。
3. 夜视或影斑视觉第一次实用。
4. 章尾让赵启明逼近。

第 3 章：主管的影子跪了

1. 赵启明当面威胁，压迫拉满。
2. 煤球吞影斑，逼出不可抵赖的异常证据。
3. 林澈录音、备份、报警、发群，亲手完成反击。
4. 杨小北登场，把职场线推向异能局。

## 六、主角与核心角色

### 林澈

温和、能忍、细致、有底线，但不是圣父。

写法：

1. 可以怕，不能废。
2. 可以穷，不能卑微讨好。
3. 每次反击都要有自己的判断。
4. 每个副本至少发挥一次数据敏感或证据整理能力。
5. 成长方向：底层社畜 -> 异常协作者 -> 城市异常清道夫。

### 煤球

小名煤球，真实代号夜阙，S 级异能兽“影噬兽”。

写法：

1. 不说人话，用眼神、尾巴、爪子、耳朵、吐出物表达。
2. 高冷、嘴硬、护短、挑食、记仇、爱踩键盘。
3. 可爱动作必须推动剧情：踩键盘触发线索，炸毛提示危险，叼晶核表达态度。
4. 吞噬越强，越虚弱、嗜睡、失控。
5. 它不是外挂，是伙伴。林澈尊重它，它才选择他。

### 杨小北

异能局外勤监察员。冷静、专业、行动力强，习惯按规则判断风险。

写法：

1. 前期带来审查、压力和秩序，不急着恋爱。
2. 对林澈的信任必须靠事件积累。
3. 感情线慢热：欣赏 -> 信任 -> 并肩 -> 主动靠近。
4. 她的靠近方式是担责、给选择权、并肩作战，不撒娇卖弱。

### 许清妍

林澈白月光，温柔体面，有距离感，身上疑似寄生“愿虫”。

写法：

1. 先保留滤镜，再让异常一点点破坏滤镜。
2. 她是否知情要悬着，不能过早定性。
3. 她的线服务林澈情感清醒：执念 -> 心疼 -> 怀疑 -> 放下。
4. “愿虫”只放大好感、愧疚和保护欲，不能写成低俗操控。

### 周言

大学室友，自由摄影师兼探店博主。嘴碎、讲义气、胆小但关键时刻不掉链子。

写法：

1. 负责烟火气、跑腿、藏猫、打听消息和轻喜剧。
2. 中期因拍到异常边角料被卷入。
3. 他是普通人视角，提醒林澈别被异能世界吞掉正常生活。

### 陈野与秦衡

陈野：黑市情报贩子，嘴碎、贪财、胆小、讲规矩。每次帮忙都要有价格或风险。

秦衡：研究所负责人，表面温和，实则想管控煤球。逻辑是“危险必须被关起来”，不能写成单薄坏人。

## 七、能力与设定

煤球能力：影噬返还。

返还物：

1. 影晶：钱、升级、装备能源，也是被盯上的原因。
2. 记忆碎片：证据线索，不能凭空造假。
3. 技能毛球：一次性能力，必须有反噬。
4. 异常残页：中后期揭开煤球、林晚和幕后组织。

限制必须写：

1. 煤球只能吞真实恶意留下的痕迹。
2. 强污染会让煤球虚弱或失控。
3. 林澈用技能毛球要付出疲惫、情绪污染等代价。
4. 影晶交易会留下波动，引来异能局或黑市。
5. 煤球愿不愿出手，取决于林澈的选择。

## 八、爽点规范

核心爽点：

> 坏人藏得越深，煤球吐得越狠；证据越具体，打脸越解气。

爽点必须落到实物和行动：

1. 钱：欠薪、赔偿、悬赏、影晶估价。
2. 证据：录音、账本、监控、转账链、记忆碎片。
3. 能力：夜视、兽语、危险预感、影遁。
4. 身份：临时协作者、交易额度、异能局登记。
5. 情感：煤球护主、林澈救猫、林晚听见猫说话。

禁止：

1. 只写反派震惊，不给证据链。
2. 只写获得能力，不展示用途。
3. 让煤球随便吐万能答案。
4. 林澈全程旁观，让猫替他完成全部反击。

## 九、幽默规范

幽默要短、准、轻，不破坏危机。

常用笑点：

1. 煤球高冷，但行为很猫。
2. 林澈很穷，煤球很挑。
3. 异能局很严肃，煤球不配合。
4. 周言嘴碎，陈野怂。

避免：

1. 大段段子。
2. 生死危机里无故耍贫。
3. 反复玩同一个梗。
4. 把林澈写油。

## 十、场景与语言

重要场景三件套：

1. 现实细节：潮湿墙皮、旧工牌、催租纸条、消毒水味。
2. 异常细节：黑气、红印、影斑、镜中延迟、动物无声盯视。
3. 角色反应：林澈握紧手机，煤球炸毛，反派停顿。

语言要求：

1. 句子顺，段落短，动作在前，解释在后。
2. 少写大段心理活动，多用行为表现情绪。
3. 不堆华丽比喻，不写晦涩长句。
4. 设定只在角色需要知道、剧情需要推进时出现。

## 十一、反派与合规

反派层级：

1. 前期：现实小人，扣工资、栽赃、威胁、钻规则漏洞。
2. 中期：异常获利者，黑市、邪性觉醒者、污染物收藏家。
3. 后期：制造影斑生态的人，试图人工制造可控 S 级异能兽。

尺度：

1. 犯罪和黑市不写教程。
2. 动物伤害不猎奇，重点写救援、揭露、惩治。
3. 不低俗擦边，不拿疾病和创伤当笑点。
4. 可以写机构里的坏人，不污名化整个职业群体。

## 十二、每章自检

写完一章，检查：

1. 这一章有没有明确冲突？
2. 林澈有没有主动做事？
3. 煤球有没有不可替代的动作？
4. 是否兑现上一章期待？
5. 是否有具体爽点或具体线索？
6. 是否留下明确章尾钩子？
7. 有没有过长设定说明？
8. 猫味、现实痛点、异能异常，至少出现两项了吗？

不满足以上 6 项，优先重修节奏。
