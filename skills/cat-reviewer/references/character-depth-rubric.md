# 人物立体度评判法

用于 `cat-reviewer` 审稿时判断角色是否立体丰满。目标不是把番茄网文写成慢节奏文学小说，而是在快节奏爽文里让人物有欲望、有矛盾、有行动、有代价。

## 来源基准

此评判法综合以下常见写作理论与经验，并改造为本书可执行检查表：

- E. M. Forster 的 flat/round character 思路：角色是否只由一个标签驱动，还是能在压力下显出复杂性与可信变化。
- Lajos Egri 的三维人物思路：生理、社会、心理三层共同塑造角色行为。
- 写作教学中常用的 characterization 原则：角色通过行动、语言、选择、关系和冲突被呈现，而不是只靠作者说明。
- 角色弧线经验：角色的 want/need、误信念、伤口、选择代价和关系反应共同推动成长。

参考链接：

- [Oregon State Guide to Characterization](https://liberalarts.oregonstate.edu/wlf/what-characterization)
- [Purdue OWL: Creative Writing](https://owl.purdue.edu/owl/subject_specific_writing/creative_writing/index.html)
- [Encyclopaedia Britannica: E. M. Forster](https://www.britannica.com/biography/E-M-Forster)

## 十项评判维度

每个重要角色按 0-2 分粗评即可。0=缺失，1=有但弱，2=清楚且能推动剧情。不是每章都要满分，但核心角色连续出场时不能长期低分。

| 维度 | 审稿问题 | 本书应用 |
|---|---|---|
| 欲望 | 角色当下想要什么？ | 林澈要证据、钱、保护煤球/林晚；杨小北要控制风险；陈野要赚钱且保命。 |
| 压力 | 什么现实或异常力量逼他行动？ | 欠薪、医药费、收容压力、影晶暴露、污染升级。 |
| 恐惧 | 他最怕失去什么？ | 林澈怕妹妹和煤球出事；煤球怕研究所/失控；杨小北怕放任风险伤人。 |
| 价值观 | 他坚持什么底线？ | 林澈心软但不糊涂；杨小北守规则；秦衡相信控制危险是必要代价。 |
| 矛盾 | 他身上有没有互相拉扯的两面？ | 能忍但会补刀，护短但怕失控，专业但会被现场事实改变判断。 |
| 主动性 | 他有没有主动选择，而非只被剧情推着走？ | 林澈必须查证、备份、报警、谈判、布局；猫提供入口但不替他完成全部。 |
| 代价 | 他的选择让他付出什么？ | 体力、情绪污染、身份风险、关系压力、钱、监管记录。 |
| 关系反应 | 他对不同人是否呈现不同侧面？ | 林澈对煤球温柔、对赵启明克制反击、对杨小北讲证据、对周言能卸下防备。 |
| 独特表达 | 语言、动作、习惯是否可辨认？ | 煤球用尾巴/爪子/叼吐物，杨小北短句职业化，周言嘴碎生活化。 |
| 变化 | 经历本章后，他的处境、认知或关系是否微变？ | 至少一个核心角色要有信息、信任、风险、能力或情绪上的推进。 |

## 人物描写专项检查

### 心理活动

合格线：

1. 重要压力点后有短心理反应，说明角色如何理解这件事。
2. 心理活动服务性格，不只写“震惊、愤怒、害怕”。
3. 心理和动作连在一起：刺激 -> 身体反应 -> 内心判断 -> 行动选择。

扣分信号：

- 大事发生后角色只行动，没有任何内心过滤，像任务脚本。
- 心理活动全是通用词，换给任何角色都成立。
- 连续大段自我剖析拖慢节奏。

修法：

- 给林澈加“证据/风险/妹妹/煤球”的判断。
- 给杨小北加“规则/风险/责任”的判断。
- 给周言加“先怕一下但还是做”的反应。
- 给反派加“怕败露、怕失控、怕上级追责”的内心或外显痕迹。

### 语言辨识度

合格线：

1. 遮住人名，也能大概分辨是谁在说话。
2. 对话里有角色关注点：林澈关注证据，杨小北关注流程，陈野关注价格，秦衡关注风险控制。
3. 对话承担关系变化，不只是传递情报。

扣分信号：

- 每个人都像作者在解释设定。
- 每个人都同样冷静、同样会吐槽、同样说狠话。
- 对话只问答信息，没有试探、隐瞒、压迫、退让或误解。

修法：

- 重写其中一句，让它带角色词汇和利益。
- 给同一句信息换表达：杨小北说“按流程”，陈野说“这价钱不对”，周言说“我先声明我怕，但我去”。

### 人物弧光

合格线：

1. 每章至少一个核心人物发生微变化。
2. 变化可以很小：多信一点、多怕一点、多承担一点、多怀疑一点。
3. 弧光必须来自选择和代价，不来自旁白宣布。

扣分信号：

- 章节结束时人物关系、认知、底线都原地不动。
- 主角只获得信息，没有承担风险。
- 配角只完成工具功能，没有留下态度变化。

修法：

- 在章尾前加一个选择：发不发、救不救、说不说、信不信、交不交。
- 让选择带代价：暴露位置、欠人情、被监管、损失钱、关系变紧张。

## 角色类型专项标准

### 主角：林澈

合格线：

1. 可以被压，但不能只挨打。
2. 每章至少做一个主动判断或选择。
3. 他的细致、数据敏感、证据整理能力要通过动作展示。
4. 心软要带来风险或代价，不能变成圣父。
5. 反击必须有他的手：录音、备份、报警、谈判、发群、钓鱼、布局。

高分表现：

- 他在害怕时仍做出有策略的动作。
- 他尊重煤球，不把煤球当工具。
- 他为了林晚和煤球承担后果，而不是等外部强者兜底。

扣分信号：

- 一章里林澈只是惊讶、被救、被解释。
- 证据链全靠煤球吐出，林澈没有验证和整理。
- 受辱后只靠反派自爆爽，主角没有推进。

### 伙伴萌宠：煤球 / 夜阙

合格线：

1. 不说人话。
2. 猫动作必须推动剧情或揭示态度。
3. 强大后必须有虚弱、嗜睡、失控、挑食、警惕等代价或反应。
4. 它有选择权，不是随叫随到的工具。

高分表现：

- 猫性和 S 级危险同时存在。
- 可爱动作改变线索、节奏或关系。
- 它护短但不完全可控，让读者既爱又担心。

扣分信号：

- 只当自动吐宝机。
- 像人一样解释设定。
- 每次都零成本解决危机。

### 秩序角色：杨小北

合格线：

1. 先按规则和风险判断，不为好感放弃职业性。
2. 对林澈的信任必须由事件累积。
3. 她的帮助要有程序、责任或代价。

高分表现：

- 她与林澈不是谁压倒谁，而是规则与现场判断互相校正。
- 她给选择权、担责、并肩作战，慢慢靠近。

扣分信号：

- 一出场就无条件相信林澈。
- 只负责解释异能局设定。
- 为暧昧牺牲职业判断。

### 现实/异常反派

合格线：

1. 反派必须知道自己靠什么获利。
2. 反派行为要能解释，不只为了让主角打脸。
3. 反派的恶意要留下可追踪痕迹。

高分表现：

- 反派的自保逻辑、利益链、恐惧都清楚。
- 被打脸不是靠降智，而是被主角抓住真实漏洞。

扣分信号：

- 只会威胁、震惊、嘴硬。
- 没有利益动机或行动成本。
- 证据突然从天而降。

### 辅助角色

合格线：

1. 每次出场至少完成一个功能：信息、烟火气、压力、冲突、喜剧、规则、情感锚点。
2. 不能长期只重复标签。

本书重点：

- 周言：普通人视角、跑腿、藏猫、轻喜剧、提醒林澈保留正常生活。
- 陈野：情报入口但收费、有风险、讲规矩。
- 秦衡：理念反派，有可辩护逻辑，不是单薄坏人。
- 许清妍：保留滤镜和疑云，不早早定性。

## 审稿输出模板

角色立体度可用这个格式：

| 角色 | 当前功能 | 立体度评分 | 问题 | 强化动作 |
|---|---|---:|---|---|
| 林澈 | 查证/反击 | 0-2 | 是否缺主动选择 | 加一个亲手验证、备份或谈判动作 |
| 煤球 | 发现/吞噬 | 0-2 | 是否像外挂 | 加代价、拒绝、误伤风险或猫性动作 |
| 杨小北 | 秩序/审查 | 0-2 | 是否太快信任 | 加程序阻力或风险判断 |

结论要落到改稿动作，不要只写“人物不够立体”。
