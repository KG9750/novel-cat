# 第012章 技能毛球：兽语

林澈差点跪下去。

哭声不是从门外传来的。

是从那团灰白毛球里炸开的。

猫的尖叫、狗的呜咽、幼崽细碎的喘气，还有某种更低、更乱的情绪，像一盆冷水兜头浇进脑子。

他扶住医院外的路灯，耳朵里嗡嗡作响。

胃里一阵翻涌。

毛球在掌心一点点化开，像湿棉絮，又像煤球最后硬塞给他的钥匙。

林澈看向安益宠物医院的招牌。

煤球不见了。

他可以等杨小北来。

也可以现在就用这东西。

风险很明显。

前几次技能毛球只让他眼痛、疲惫，这一次光是碰到，他就已经被那些情绪冲得站不稳。

可地下室里的动物等不了。

煤球也等不了。

林澈把毛球放进口中。

它没有味道。

下一秒，尖锐耳鸣直接穿透头骨。

世界安静了一瞬，又猛地变得吵闹。

不是人声。

是动物的意思。

它们不会说完整的话，更多是气味、疼痛、恐惧和重复的简单词。

冷。

灯疼。

针苦。

黑盆会吃影子。

不要睡。

不要进唱歌的笼子。

林澈扶着墙，强迫自己呼吸。

兽语不是翻译器。

它像把几十个惊恐的梦同时塞进他脑子里。

他很快发现，自己不能一直听。只要注意力落在某一声哭里，鼻腔就会幻觉到消毒水和血腥味，连膝盖都跟着发软。他把耳机线缠在手指上，一圈一圈勒紧，逼自己每三十秒停一次，核对手机里的照片、收据编号和后巷位置。

兽语能指路，但证据不能靠痛觉作证。他需要把它们翻成警方、杨小北、程姨都能看懂的东西，否则再真实也只会被说成幻听。

林澈把备忘录标题改成“可核查线索”，才继续听。

他听见一只小狗反复想着“蓝绳、程、回家”，也听见奶猫饿到发飘的细声。更多动物只剩本能，怕灯、怕手套、怕那个会响的金属滑轨。

每一种恐惧都留下残渣。

林澈忽然想咬人。

又忽然想钻进黑暗里躲起来。

他用指甲掐住掌心，疼痛把自己拉回来。

“找入口。”他低声对自己说，“别被它们的怕带着跑。”

他蹲在路边，把听见的东西拆开。

重复三次以上的，留下。

只有一只动物喊过的，先放一边。

他以前审数据，最怕被噪音带偏。现在这些叫声也是噪音，只是噪音里夹着活命的线索。

“冷门”“蓝线”“苦水”“唱歌笼”被他写进备忘录。

写到“唱歌笼”时，林澈耳边忽然冒出一段尖细的金属摩擦声。

那声音让他膝盖一软，脑子里同时浮起几只动物被推上滑轨的画面。它们不懂机器，只记得笼子会唱，唱完就会睡。

他差点把手机摔了。

煤球给他的不是答案。

是把整个地下室的恐惧直接倒给他，让他自己筛。

筛错一次，地下室里就会少一个还能等的声音。

林澈把这句话也记下，提醒自己别被愤怒推着走。

手机震动。

杨小北回了消息：位置收到，十五分钟内到。不要单独进入异常核心。保留证据，优先撤离普通人。

十五分钟。

林澈看了一眼医院后门。

黑袖男人带走煤球，绝不会等十五分钟。

他没有从正门进。

他绕到后巷，找到B1通风口。风口边缘有三道抓痕，一横两竖。

煤球留下的。

林澈蹲下来，手指摸过抓痕。

耳鸣里忽然多了一声低低的哈气。

不是人话，也不是完整猫语。

更像一团压着痛的情绪：别信亮灯的门。

林澈鼻子一酸。

“知道了。”

后巷堆着医疗废物箱。

他没有乱翻，只拍照记录编号。箱盖缝里渗出淡淡腥甜味，兽语状态下，那味道变成了动物记忆里的一个词：苦水。

苦水之后，是睡。

睡了就被推走。

旁边还有一只破旧猫包。

猫包拉链断了，内侧粘着几根橘色短毛。兽语状态下，那些毛发残留着很浅的情绪：等、疼、主人没有来。

林澈把猫包拍下来，发给程姨确认。

程姨很快回了消息：这不是我的，是另一个救助人的猫包。她家猫上个月在安益说“术后感染死亡”。

又一条线。

这家医院吞掉的，不止今晚这一批。

他把这条线索单独标红，备注：历史死亡记录需查。

林澈沿着墙根找。

后门刷卡器旁，墙面有一块颜色更深的瓷砖。普通人看只是潮，兽语带来的混乱感却让那处不断冒出“进去、下去、冷”的念头。

他用程姨发来的照片对比。

照片里的铁架背景，右侧有一道蓝色安全线。

后门里，也有同样的蓝线。

林澈拍下对比图，发给杨小北和周言。

周言秒回：你半夜给我发宠物医院后门干啥？你不会又捡猫了吧？

林澈发过去一个压缩包链接：别问，先帮我备份。要是我十分钟不回，把文件发给我标好的三个邮箱。

周言：？

周言：你这语气像遗书，能不能别吓我？

林澈没有解释。

他听见里面的笼子又响了。

蓝绳小狗的情绪最清晰：黑猫在冷门后，黑猫咬疼的线，黑猫睡不醒。

还有一个很弱的念头，来自三只奶猫中最小的那只。

它记不住路，只记得黑猫靠近时，笼子外面那团会吃影子的东西缩了一下。

煤球还活着。

只是被什么困住了。

冷门后。

冷库后面还有门。

林澈找来后巷的废弃木板，搭到通风口下方，没爬进去，只把手机摄像头伸进去。

画面抖动着扫过冷库。

地上有粉末、倒笼、几滴黑色唾液。

冷库最里面，废弃药品柜旁，有一条滑轨。

滑轨后方不是墙，是一扇贴着银色保温板的暗门。

煤球就在那后面。

林澈刚收回手机，耳鸣突然变尖。

几十只动物的声音同时停了一下。

那种停顿比哭叫更吓人。

紧接着，一个共同的情绪从地下涌上来。

不是求救。

是恐惧到极点的提醒。

它醒了。

后门里的灯，一盏接一盏亮起。

每一盏灯下，都拖出一条细长的黑影，朝冷库深处爬去。
